# three-js-mini-solar-system

Develop Status: InProgress

# Build Setup

```
# install dependencies
$ yarn

# launch server
$ yarn dev:backend
$ yarn dev:frontend

# build for production and launch server
$ yarn build:backend
$ yarn build:frontend
```
